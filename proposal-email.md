I would like to use the Kalman filter applications topic for my course project.

Specifically, I plan to implement an Extended Kalman Filter (EKF) for offline rocket flight state estimation. The estimator will fuse logged IMU and barometric sensor data to reconstruct altitude, vertical velocity, and accelerometer bias from real high-power rocket flights, using publicly available data from the [DOFPro flight data archive](https://dofpro.org/RCK/fltdata/FlightData.html).

The core papers I will be referencing are:

- [A New Approach to Linear Filtering and Prediction Problems](https://doi.org/10.1115/1.3662552) (R.E. Kalman, 1960)
- [Extended Kalman filter synthesis for integrated global positioning/inertial navigation systems](<https://doi.org/10.1016/S0096-3003(98)10068-1>) (F.A. Faruqi & K.J. Turner, 2000)
- [Adaptive Kalman Filtering for INS/GPS](https://doi.org/10.1007/s001900050236) (A. H. Mohamed & K. P. Schwarz, 1999)
- [PX4 EKF2: A State Estimation Framework for Autonomous Drone Flight](https://docs.px4.io/main/en/advanced_config/tuning_the_ecl_ekf.html) (P. Riseborough & PX4 Contributors, 2019)
